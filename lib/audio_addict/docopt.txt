AudioAddict Radio

Usage:
  radio set CHANNEL [NETWORK]
  radio channels [SEARCH]
  radio current
  radio vote
  radio (-h|--help|--version)

Commands:
  set
    Save the AudioAddict channel and optionally the network to the 
    config file for future use. 

  channels
    Show list of channels.

  current
    Show currently configured network and channel, and the currently 
    playing track.

  vote
    Vote on the currently playing track.

Params:
  NETWORK
    AudioAddict network key. Can be one of:
    di, rockradio, radiotunes, jazzradio or classicalradio

  CHANNEL
    AudioAddict channel key.

  SEARCH
    Channel name or a partial name to search for.

Examples:
  radio set modernrock rockradio
  radio set modernrock
  radio channels
  radio channels metal
